SupportedPlatforms
==================

Supported Platforms
===================

Redis can be compiled in most POSIX systems, but the development
targets mainly:

-  Linux
-  Mac OS X
-  FreeBSD
-  OpenBSD
-  Solaris (startting with `Version 1.1 <RoadMap.html>`_)

Windows (using CygWin) is not a supported platform.
