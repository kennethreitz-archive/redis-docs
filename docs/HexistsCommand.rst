HEXISTS \_key\_ \_field\_ (Redis >
==================================

1.3.10)= *Time complexity: O(1)*
    Return 1 if the hash stored at *key* contains the specified
    *field*.

    Return 0 if the *key* is not found or the *field* is not present.
